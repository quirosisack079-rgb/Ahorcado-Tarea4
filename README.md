Juego de Ahorcado

Es un juego donde el programa elige una palabra al azar y el jugador tiene que adivinarla letra por letra. Si falla 5 veces pierde.

Como funciona


El programa elige una palabra al azar de una lista
Muestra guiones según la cantidad de letras que tiene
El jugador ingresa una letra
Si la letra está en la palabra, reemplaza el guion
Si no está, pierde una vida
Gana si adivina todas las letras, pierde si llega a 0 vidas
